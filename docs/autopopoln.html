Инструкция по работе с программой АвтоПополнение

При первом запуске откроется вкладка "Настройки". Программа попытается автоматически найти где установлен ©КонсультантПлюс. Она ищет в "C:\Consultant" и если не находит, ищет ярлык с Консультантом на рабочем столе из которого берёт путь к папке.

В настройках нужно ввести название организации, путь к папке с Консультантом, путь к папке с обновлениями, указать где находится STT файл и выбрать куда складывать STT и USR файлы.

[x] - Вместо x подставляется буква диска с которого была запущена программа.

Параметр "Закрывать программу" нужен для того, чтобы при запуске с ключом /AUTO программа запускалась, выполняла все выбранные функции(копирует обновления, служебные файлы, запускает ©КонсультантПлюс с ключами для обновления) и закрывалась.

После того как вы всё настроите, в разделе "Базы" будет показан список установленных у клиента баз.

Здесь можно выбрать базы для которых нужно копировать обновления.

В разделе "Пополнение" можно проконтролировать версии файлов cons.exe и *.res. Также там отображается свободное место на диске где установлен ©КонсультантПлюс.

Также там отображается текущая дата и время на компьютере клиента. Её можно изменить в поле ввода с помощью клавиатуры или с помощью колёсика мышки. Если программа была запущена с правами администратор, при нажатии на кнопку Изменить программа сама изменит дату и время, а если нет, запустит системные настройки по изменению даты и времени.

При нажатии на кнопку Копировать программа выполнит выбранные в настройках действия - скопирует обновления с флешки и служебные файлы USR и STT на флешку.

Кнопка Выполнить запускает ©КонсультантПлюс с ключами для принятия скопированных пополнений.

Кнопка Папка К+ запустит проводник с открытой папкой где установлен Консультант.

В разделе "Отчёт" отображаются даты последних 10 обновлений и содержимое файла last_rec.txt.

Если вы создали файл bases.lst в программе EditBases, то в разделе "Документация" будут показаны ваши RTF файлы.
