Инструкция по работе с программой EditBases

Описания баз можно создать вручную. В левый столбец вписываете краткое название, в правый полное. Новую строку можно добавить с помощью кнопки Insert или из контекстного меню, либо из меню "Правка".

Также можно загрузить описания из текстового файла вот такого формата: LAW=Версия Проф

Описания ошибок и ключей командной строки загружаются из RTF файла.

После того как всё сделаете, не забудьте сохранить в файл bases.lst. Этот файл потом нужно скопировать в ту папку где находится программа АвтоПополнение.
